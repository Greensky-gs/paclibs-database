# Paclibs-database

Database for my [pacboy](https://github.com/Greensky-gs/pacboy) utility

## How it works

1. Use [pacboy](https://github.com/Greensky-gs/pacboy)
2. Run `pacboy --change-config`, and in the "new repo url" field, provided either this one, or the one you own.
3. **Important** The repo you use **must** have a `mirror` branch, on which everything will be cloned, and left as is, so typically the same as main, but without the README
